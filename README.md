# Recovery Rebuilt Website

This is a custom React + Vite website for Recovery Rebuilt.

## Local setup

1. Install Node.js.
2. Open this folder in a terminal.
3. Run:

```bash
npm install
npm run dev
```

## Build for deployment

```bash
npm run build
```

The finished static website will be created in the `dist` folder.

## Recommended deployment options

- Vercel: Import the GitHub repository, choose Vite, and deploy.
- IONOS Deploy Now: Connect the GitHub repository and deploy as a React / single-page app.

## Things to customize before launch

- Replace `info@recoveryrebuilt.com` if needed.
- Add a real business phone number.
- Replace the contact form with a production form service such as Formspree, Netlify Forms, or a backend contact form.
- Replace the placeholder RR mark with your finalized vector logo when ready.
